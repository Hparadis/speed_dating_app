# Back End of our app
## Tools we will be using:
| Tool     | Purpose                        |
| -------- | ------------------------------ |
| Express  | Backend framework              |
| Mongoose | MongoDB ORM                    |
| JWT      | Authentication                 |
| bcryptjs | Password hashing               |
| dotenv   | Environment variables          |
| cors     | Frontend/backend communication |
| nodemon  | Auto-restart dev server        |
